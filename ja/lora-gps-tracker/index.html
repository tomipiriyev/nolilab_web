テクノロジー

LoRa とは何ですか? LoRa によって GPS トラッカーがより優れたものになるのはなぜですか?

LoRaはロングレンジの略です。これは、Semtech によって開発された無線変調技術であり、チャープ スペクトラム拡散 (CSS) を使用して、Wi-Fi、Bluetooth、または Zigbee では不可能な距離で小さなデータ ペイロードを送信します。

重要な物理学: LoRa は、データ レートと引き換えに範囲と貫通力を実現します。データの送信は遅いですが、壁、木、地形を越えて非常に遠くまで、非常に確実に送信します。位置更新が数秒から数分ごとに送信されるデータの小さなパケットである GPS トラッキングの場合、これは理想的な一致です。

範囲

キロメートルスケールの距離

LoRa は、開けた地形で 20+ km を達成します。 Wi-Fi は 100 メートルをカバーします。 Bluetoothは50mをカバーします。同等の出力レベルで LoRa の通信範囲に匹敵するライセンスフリーの無線テクノロジーは他にありません。

力

超低エネルギー消費

LoRa 無線は、送信中にミリワット、スリープ中にマイクロワットを消費します。 Loko は、長いアップデート間隔と組み合わせることで、1 年間のバッテリー寿命を実現します。これは、携帯電話や Wi-Fi 無線では不可能です。

浸透

障害物を乗り越えて移動する

LoRa のサブ GHz 周波数 (EU では 868 MHz、米国では 915 MHz) は、高周波数での 2.4 GHz Wi-Fi やセルラーよりもはるかに優れて、植生、建材、穏やかな地形を透過します。

独立

ネットワークインフラがない

LoRa P2P comm デバイス間を直接接続します。追跡チェーンのどの時点でも、携帯電話の基地局、ゲートウェイ、サーバー、インターネット接続は必要ありません。

料金

運用コストゼロ

LoRa はライセンスフリーの ISM 無線帯域を使用します。通信料金やデータ料金はかからず、通信事業者も必要ありません。費用はハードウェアの費用だけです。

信頼性

単一障害点がない

LoRa P2P リンクには、トラッカーとレシーバーという 2 つのコンポーネントがあります。オフラインになって接続が切断される可能性のあるクラウド、キャリア ネットワーク、サーバーはありません。

技術的パラメータ

LoRa には、範囲、更新頻度、消費電力の間で直接のトレードオフを生み出す構成可能な変調パラメータがあります。これらを理解すると、特定の使用例に合わせて Loko を構成するのに役立ちます。

拡散率 (SF) SF7 – SF12 SF が高い = 航続距離が長く、更新速度が遅くなり、パケットあたりのバッテリー使用量が増加します。 SF12 は最大範囲を提供します。

位置精度

GPS の精度: LoRa との関係 (何もありません)

comm に関する誤解: LoRa 無線は GPS の位置精度にまったく影響しません。 2 つのシステムは完全に独立しています。

GPS の精度 はすべて Loko Air の GNSS 受信機から取得されます。 Loko は、4 つの衛星コンステレーションを同時にサポートします。

  • GPS (米国) — 衛星 31 基
  • GLONASS (ロシア) — 衛星 24 基
  • Galileo (EU) — 衛星 30 基
  • BeiDou (中国) — 35 個以上の衛星

4 つの星座をすべて同時に使用すると、特に一部の衛星しか見えない峡谷の地形や密集した林冠などの困難な環境において、Loko の位置取得 (初回測位までの時間) が速くなり、精度が向上します。

オープンスカイでの一般的な位置精度: 2～5メートルのCEP (循環エラーの可能性があります)。これは、任意のマルチコンステレーション民生用 GNSS デバイスと同一であり、位置の送信にどの無線技術が使用されるかには依存しません。

LoRaの役割 は純粋に送信です。GPS 座標を取得し、それを長距離にわたってエラーなく受信機に送信します。 LoRa は、前方誤り訂正 (コーディング レート パラメーター) を使用して、ノイズの多い無線環境でも位置データが完全な状態で到着することを保証します。

よくある質問

Common 長距離 LoRa GPS トラッカーに関する質問

LoRa GPS トラッカーはどこまで到達できますか?

Loko は、視界がクリアな開けた地形で 20+ km を達成します。都市環境や密林では、実用的な範囲は 1 ～ 5 km です。受信機を高くすると通信範囲が大幅に広がります。丘の頂上や車の屋根に設置すると、有効カバーエリアが大幅に広がります。を参照してください。 GPS および LoRa テクノロジーのページ 詳細な伝播情報については、

LoRa GPS トラッカーとは何ですか?

LoRa GPS トラッカーは、GNSS 受信機 (衛星から位置を決定する) と LoRa 無線送信機 (その位置を長距離にわたってワイヤレスで送信する) を組み合わせています。セルラー GPS トラッカーとは異なり、LoRa GPS トラッカーは SIM カードやモバイル ネットワークを必要としません。トラッカー comm は無線を介して近くの受信機と直接通信します。

LoRa GPS トラッカーにはサブスクリプションが必要ですか?

Loko にはサブスクリプションは必要ありません。携帯電話ネットワークではなく LoRa 無線を使用するため、SIM カード費用、データプラン料金、月額料金はかかりません。ハードウェアの料金を一度支払うだけで、無期限に追跡できます。これは、セルラー追跡ソリューションに対する LoRa の主な利点の 1 つです。

GPS トラッキングにおける LoRa P2P と LoRaWAN の違いは何ですか?

LoRa P2P (ピアツーピア) は、直接デバイス間の communication です。トラッカーは間にインフラストラクチャを介さずに受信機と直接通信します。 LoRaWAN はゲートウェイとネットワーク サーバーを使用し、ゲートウェイが展開されているエリアでより広い範囲をカバーしますが、そのインフラストラクチャが存在する必要があります。 Loko は LoRa P2P を使用します。つまり、2 台のデバイスを超えたセットアップなしでどこでも動作します。完全な比較については、 LoRaWANページ.

GPS トラッキングでは LoRa がセルラーよりも優れていますか?

それはアプリケーションによって異なります。 LoRa は、携帯電話の通信範囲のないエリアで追跡が必要な場合、月額料金が不要な場合、複数年のバッテリー寿命が必要な場合、またはデバイスごとのサブスクリプション費用なしで複数の資産を追跡する場合に適しています。近くに受信機がなく、追跡対象のアイテムが通信範囲内に留まり、世界中に到達する必要がある場合は、セルラーの方が適しています。屋外、遠隔地、および農業アプリケーションの場合、ほとんどの場合、LoRa が総所有コストの点で優れています。

LoRa GPS トラッカーはどのくらい正確ですか?

Loko の GPS 精度は、LoRa 無線ではなく、完全に GNSS 受信機から得られます。 Loko は、GPS、GLONASS、Galileo、および BeiDou を同時にサポートし、オープンスカイで約 2 ～ 5 メートル CEP の位置精度を提供します。これは、マルチコンステレーション GNSS デバイスと同じです。 LoRa 無線は、位置誤差を引き起こすことなくこの位置を送信します。

LoRa GPS トラッカーの範囲に影響を与える要因は何ですか?

主な要因は、受信機の標高 (標高が高いほど通信距離が長くなります)、地形 (見通し内であれば最大通信距離が得られますが、丘や密林によって通信距離が減少します)、拡散率 (SF が高い = 通信距離は長いですが、更新が遅くなります)、送信電力、およびアンテナ品質です。高いレシーバーを備えた開けた地形では、Loko は 20+ km を達成します。

LoRa GPS トラッカーは複数のデバイスを同時に追跡できますか?

はい。 Loko は、単一の Loko Ground レシーバーに同時に接続された複数の Loko Air トラッカーをサポートします。追跡されたすべての位置は同じオフライン マップ上にリアルタイムで表示されます。これは、単一のデバイスから動物の群れ、人々のチーム、または車両群を追跡するのに役立ちます。
