Loko GPS トラッカー セットアップ ガイド

Loko 航空および地上ユニットを設定し、モバイル アプリに接続し、数分でオフライン GPS 追跡を開始するためのステップバイステップの手順。

クイックスタートガイド

Loko GPS トラッカーを数分で操作できるようになります。

  1. 開梱して充電:付属の USB-C ケーブルを使用して航空ユニットと地上ユニットの両方を完全に充電します (約 60 分かかります)
  2. アプリをダウンロード:iOS App Store または Google Play ストアから Loko アプリをインストールします
  3. 電源オン:両方のユニットの電源ボタンを 3 秒間押し続けます
  4. ペアユニット:アプリを開き、ペアリング ウィザードに従って航空ユニットと地上ユニットを接続します
  5. 設定を構成します:更新間隔、ジオフェンス境界、通知設定を設定する
  6. マウントトラッカー:付属の取り付け金具を使用して航空ユニットを資産にしっかりと取り付けます

高度なセットアップ: LoRaWAN 接続の構成

Loko を LoRaWAN ネットワークに接続して、カバレッジを拡張します。

20km の P2P 範囲を超える接続が必要なアプリケーションの場合は、The Things Network (TTN) または他の LoRaWAN プロバイダーに接続するように Loko を構成します。

  1. DevEUI にアクセスします。Loko アプリの [デバイス情報] で一意のデバイス ID を見つけます。
  2. アカウントを作成する：console.thethingsnetwork.org またはお好みの LoRaWAN プロバイダーに登録してください
  3. デバイスの登録:OTAA (推奨) または ABP アクティベーションを使用して Loko を追加します
  4. キーを構成します。Loko アプリの設定に AppEUI、AppKey、NwkKey を入力します。
  5. テスト接続:TTN コンソールで最も近いゲートウェイへの接続を確認する
  6. パラメータを調整します:SF、BW、DR 設定を所在地に合わせて最適化する

注: LoRaWAN のセットアップには、近くのゲートウェイからのカバレッジが必要です。 LoRaWAN がカバーされていない地域では P2P モードを使用してください。

一般的な問題のトラブルシューティング

一般的なセットアップと操作の問題の解決策:

接続の問題

ユニットが接続されていない場合: バッテリーレベルを確認し、両方のユニットが 20km の範囲内にあることを確認し、同じ周波数帯域設定を確認して、必要に応じて両方のユニットを再起動します。

GPSロックの問題

GPS が取得できない場合: 晴れた空の景色を確認し、アンテナの向きを確認し、コールド スタートまで最大 60 秒待って、ファームウェアが更新されていることを確認します。

アプリの接続性

アプリが地上ユニットに接続しない場合: 位置情報サービスを有効にし、Bluetooth の許可を確認し、10 メートルの範囲内にいることを確認して、携帯電話の Bluetooth を再起動します。
