Outil de Configuration GPS Tracker Loko

Comment utiliser l'outil de configuration Loko

L'Outil de Configuration en Ligne Loko vous permet de configurer les unités Loko Air et Ground directement depuis votre navigateur en utilisant l'API Web Serial — aucune installation de logiciel requise. Vous avez besoin de Google Chrome (bureau) et d'un câble USB pour connecter votre appareil.

Connexion étape par étape

  1. Ouvrez cette page dans Google Chrome sur un ordinateur de bureau (Windows, macOS ou Linux).
  2. Allumez votre unité Loko Air ou Ground.
  3. Connectez l'appareil à votre ordinateur avec un câble USB-C.
  4. Sélectionnez l'onglet correct (Unité Air ou Unité Ground) dans l'outil ci-dessus.
  5. Cliquez sur Connecter, puis choisissez le port série correct dans l'invite du navigateur.
  6. L'outil lit automatiquement les paramètres actuels de l'appareil.
  7. Modifiez les paramètres souhaités et cliquez sur Appliquer pour les enregistrer sur l'appareil.

Paramètres configurables — Unité Air

L'Unité Air est le tracker GPS attaché à l'actif que vous souhaitez suivre. Les paramètres suivants peuvent être configurés :

  • ID Appareil 1 & ID 2 — Identifiants uniques qui apparient l'Unité Air avec une Station Ground. Les ID doivent correspondre entre les appareils appariés.
  • Fréquence radio (Hz) — Fréquence de transmission LoRa. Doit correspondre à la Station Ground. Bandes par défaut : 868 MHz (UE), 915 MHz (US), 923 MHz (AS).
  • Intervalle de mise à jour GPS (secondes) — Fréquence à laquelle l'Unité Air transmet sa position GPS. Des intervalles plus courts augmentent la consommation de batterie. Plage : 1–3600 secondes.
