Guide de configuration LoRaWAN pour le traqueur GPS Loko | Intégration TTN

Connectez votre tracker GPS Loko à The Things Network (TTN) avec ce guide de configuration LoRaWAN étape par étape. Configurez les passerelles, les appareils et démarrez le suivi GPS IoT.

1-7

Configuration initiale de la passerelle

Étape 1 :Créez un compte sur le site Web The Things Network.

Étape 2 :Accédez à la console via l'icône de profil.

Étape 3 :Sélectionnez la région appropriée pour votre emplacement.

Étape 4 :Configurez votre passerelle selon les spécifications du fabricant.

Étape 5 :Enregistrez votre passerelle avec le plan de fréquences approprié pour votre région.

Étape 6 :Utilisez l'ID de passerelle de la configuration de votre fabricant.

Étape 7 :Terminez le processus d’enregistrement de la passerelle.

Important:
Assurez-vous de choisir le plan de fréquences approprié en fonction de votre région pour garantir une bonne connectivité.
8

Créer une candidature

Cliquez sur le bouton "Créer une application" et donnez un Nom/ID à votre application.

9

Enregistrer manuellement le périphérique final

Enregistrez manuellement un appareil final. Pour le JoinEUI, vous pouvez mettre n'importe quoi. Cela correspond aux paramètres APP-EUI sur l'application Loko Configuration Tools.

10

Générer les informations d'identification de l'appareil

Cliquez sur le bouton Confirmer. Ensuite, d'autres paramètres apparaîtront. Vous pouvez simplement appuyer sur les boutons de génération afin de créer Dev-EUI et App-Key.

11

Récupérer les paramètres d'activation

Récupérez les paramètres de l'appareil dans la vue d'ensemble. Vous aurez besoin de ces paramètres d'activation pour configurer votre unité Loko Air :

  • AppEUI (RejoindreEUI)
  • DevEUI
  • Clé d'application
12

Configurer le formateur de charge utile

Accédez aux formateurs de charge utile et choisissez CayenneLPP pour la liaison montante.

13-15

Configurer l'appareil Loko

Étape 13 :Ouvrez l'application Loko Configuration Tool sur votre appareil.

Étape 14 :Lisez les paramètres actuels, puis activez la fonctionnalité LoRaWAN.

Étape 15 :Remplissez les trois paramètres requis à partir des informations d'activation du périphérique final de TTN :

  • AppEUI (depuis TTN)
  • DevEUI (de TTN)
  • AppKey (de TTN)

Appuyez sur envoyer pour télécharger la configuration, puis redémarrez votre unité Loko Air.

Note:
Assurez-vous de copier les paramètres exactement tels qu'ils apparaissent dans la console The Things Network pour éviter les problèmes de connexion.
16

Surveiller les données en direct

Surveillez la page de l'appareil final sur The Things Network pour afficher les données en direct et l'emplacement de l'unité Loko Air sur la carte une fois que la passerelle reçoit des informations.

Vous devriez voir la transmission de données et pouvoir suivre la position de votre appareil en temps réel.
