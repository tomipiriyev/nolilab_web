Loko offre deux modes de communication puissants pour suivre vos actifs n'importe où, n'importe quand.

Communication LoRa directe entre votre traceur Loko et smartphone — aucun internet, aucun abonnement, aucun frais mensuel. Parfait pour les aventures en plein air et les emplacements éloignés.

Connectez-vous aux réseaux LoRaWAN publics ou privés pour une portée étendue et un suivi mondial. Accédez à l'emplacement de votre traceur depuis n'importe où dans le monde via le cloud.

Loko Air envoie les données à la passerelle LoRaWAN, la passerelle les transmet au serveur cloud, votre téléphone accède aux données depuis le cloud

Foire Aux Questions

Tout ce que vous devez savoir sur le Traceur GPS Loko. Vous ne trouvez pas de réponse ? Contactez notre équipe d'assistance.

Loko est le plus petit traceur GPS léger et open-source au monde, conçu pour la surveillance de localisation en temps réel. Il utilise la communication radio LoRa P2P (aucune carte SIM requise) et est idéal pour les drones, les animaux, l'équipement ou les objets dans les zones sans couverture cellulaire. Sa consommation ultra-faible permet jusqu'à 30 jours d'autonomie sur une seule charge.

Le traceur GPS Loko fonctionne avec deux unités : l'Unité Air (émetteur) se fixe à votre appareil et envoie les données GPS via radio LoRa P2P. L'Unité Ground (récepteur) reçoit les données de l'Unité Air et les transmet à une application smartphone via Bluetooth. L'application affiche les emplacements sur une carte 2D. Aucun internet ou réseau cellulaire requis pour le suivi GPS.

Le traceur GPS Loko est résistant aux chocs, résistant à l'eau et étanche à la poussière. Notez que Loko n’est pas étanche. Il n'est pas recommandé de le mettre sous l'eau. Conçu pour le suivi GPS extérieur dans des conditions difficiles.

Jusqu'à 30 unités de traceur GPS Loko Air peuvent être associées à une seule unité Ground, chacune avec un identifiant unique. Parfait pour suivre plusieurs actifs simultanément.

Le traceur GPS Loko peut s'intégrer aux passerelles LoRaWAN® existantes pour les applications IoT. Il dispose également d'une communication LoRa P2P directe pour le suivi GPS hors ligne sans dépendance d'infrastructure.
