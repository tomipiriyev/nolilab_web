Загрузка приложений и программного обеспечения Loko GPS Tracker

Загрузите приложение отслеживания Loko для iOS, Android и LineageOS, а также инструмент настройки рабочего стола. Для автономного GPS-слежения подписка не требуется.

Приложение «Локо Навигация»

Отслеживайте свои устройства Loko с высокой точностью в режиме реального времени. Особенности автономных карт, оповещений о геозонах и подробной истории отслеживания — все в одном интуитивно понятном интерфейсе. Для GPS-слежения не требуется Интернет.

Системные требования

Мобильное приложение

  • iOS 13.0 или новее
  • Android 8.0 или более поздняя версия
  • 50 МБ бесплатного хранилища

Инструмент настройки

  • macOS 10.14+
  • Windows 10/11
  • Линукс (64-разрядная версия)
  • Требуется USB-порт
