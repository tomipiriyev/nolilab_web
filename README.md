# nolilab_web — Website Source

This directory is the **Git repository** that is deployed to GitHub Pages at
[https://nolilab.com](https://nolilab.com).

> Automation scripts, documentation, and analytics data live **outside** this
> directory in the parent project. See the [project README](../README.md).

---

## Language versions

Each language subdirectory (`es/`, `fr/`, `ja/`, `ru/`, `zh/`) mirrors the
English page set. Blog posts inside each language's `blog/` subdirectory are
generated by the translation scripts — do not edit them manually.

| Code | Language | Directory |
|------|----------|-----------|
| `en` | English (default) | `/` |
| `es` | Spanish | `/es/` |
| `fr` | French | `/fr/` |
| `ja` | Japanese | `/ja/` |
| `ru` | Russian | `/ru/` |
| `zh` | Chinese (Simplified) | `/zh/` |

## Configuration files

| File | Purpose |
|------|---------|
| `_headers` | HTTP response headers (security, caching) |
| `_redirects` | URL redirects from legacy Shopify URLs |
| `robots.txt` | Search crawler directives |
| `sitemap.xml` | XML sitemap for SEO |
| `llms.txt` | Context for AI/LLM tools |
| `google7179992cd25ae374.html` | Google Search Console verification |

## Deployment

Pushes to `main` automatically deploy via the GitHub Actions workflow at
`.github/workflows/static.yml`.
