技术

什么是 LoRa 以及为什么它使 GPS 跟踪器变得更好？

LoRa 代表长距离。它是 Semtech 开发的一种无线调制技术，使用线性调频扩频 (CSS) 在 Wi-Fi、蓝牙或 Zigbee 无法实现的距离上传输小数据有效负载。

关键物理原理：LoRa 以数据速率换取范围和穿透力。它发送数据的速度很慢，但它发送的距离很远而且非常可靠，甚至可以穿过墙壁、树木和地形。对于 GPS 跟踪，位置更新是每隔几秒到几分钟发送的小数据包，这是一个理想的匹配。

为什么 LoRa 在 GPS 跟踪方面优于其他无线协议

范围

公里级距离

LoRa 在开阔地形中达到了 20+ km 的水平。 Wi-Fi覆盖范围100m。蓝牙覆盖范围50m。在相当的功率水平下，没有其他免许可无线电技术能够接近 LoRa 的范围。

力量

超低能耗

LoRa 无线电在传输期间消耗毫瓦，在睡眠时消耗微瓦。结合较长的更新间隔，Loko 的电池寿命可达 1 年——这对于蜂窝或 Wi-Fi 无线电来说是不可能的。

渗透

穿越障碍

LoRa 的 sub-GHz 频率（欧盟为 868 MHz，美国为 915 MHz）穿透植被、建筑材料和温和地形的能力远远优于高频下的 2.4 GHz Wi-Fi 或蜂窝网络。

独立

没有网络基础设施

LoRa P2P comm 直接连接设备到设备。跟踪链中的任何一点都不需要手机信号塔、网关、服务器、互联网连接。

成本

零运营成本

LoRa 使用免许可的 ISM 无线电频段。无需通话费、数据费，也无需运营商。硬件成本是您唯一的开支。

可靠性

无单点故障

LoRa P2P 链路有两个组件：跟踪器和接收器。没有云，没有运营商网络，没有可以离线并断开连接的服务器。

位置精度

GPS 精度：LoRa 与它有什么关系（无关）

common 的一个误解：LoRa 无线电对 GPS 位置精度没有任何影响。这两个系统是完全独立的。

GPS 精度 完全来自 Loko Air 中的 GNSS 接收器。 Loko 同时支持四个卫星星座：

  • GPS (美国) — 31 颗卫星
  • GLONASS （俄罗斯）— 24 颗卫星
  • Galileo (欧盟) — 30 颗卫星
  • BeiDou （中国）— 35颗以上卫星

同时使用所有四个星座使 Loko 能够更快地获取位置（首次定位时间）和更高的精度，特别是在峡谷地形或茂密的树冠等困难环境中，只有一些卫星可见。

开阔天空中的典型位置精度： 2-5米CEP （可能存在循环误差）。这与任何多星座消费 GNSS 设备相同，并且与使用哪种无线电技术来传输位置无关。

LoRa的作用 纯粹是传输：它获取 GPS 坐标并将其长距离无误地传送到接收器。 LoRa 使用前向纠错（编码率参数）来确保即使在嘈杂的无线电环境中位置数据也能完好无损地到达。

常问问题

Common 关于远程 LoRa GPS 跟踪器的问题

LoRa GPS 追踪器可以到达多远？

Loko 在开阔地形、视线清晰的情况下达到了 20+ km 的水平。在城市环境或茂密的森林中，实用范围为 1–5 km。抬高接收器可显着扩大覆盖范围 - 将其放置在山顶或车顶上可以显着增加有效覆盖面积。请参阅 GPS 和 LoRa 技术页面 获取详细的传播信息。

什么是 LoRa GPS 跟踪器？

LoRa GPS 跟踪器将 GNSS 接收器（确定卫星位置）与 LoRa 无线电发射器（长距离无线发送位置）结合在一起。与蜂窝 GPS 跟踪器不同，LoRa GPS 跟踪器不需要 SIM 卡或移动网络 - 跟踪器 comm 通过无线电直接与附近的接收器连接。

LoRa GPS 跟踪器需要订阅吗？

Loko 不需要任何订阅。由于它使用 LoRa 无线电而不是蜂窝网络，因此没有 SIM 卡费用、数据计划费用和月费。您只需支付一次硬件费用即可无限期跟踪。这是 LoRa 相对于蜂窝跟踪解决方案的主要优势之一。

LoRa、P2P 和 LoRaWAN 对于 GPS 跟踪有什么区别？

LoRa P2P（点对点）是直接的设备到设备 comm 通信 — 跟踪器直接与您的接收器通信，中间没有基础设施。 LoRaWAN 使用网关和网络服务器，可在部署网关的区域提供更广泛的覆盖范围，但需要存在基础设施。 Loko 使用 LoRa P2P，这意味着它可以在任何地方工作，无需在两个设备之外进行设置。阅读完整的比较 LoRaWAN 页面.

对于 GPS 跟踪，LoRa 是否比蜂窝网络更好？

这取决于应用程序。当您需要在没有蜂窝网络覆盖的区域进行跟踪、不希望收取月费、需要多年的电池寿命或在无需按设备订阅费用的情况下跟踪多个资产时，LoRa 会更好。当您需要在附近没有接收器的情况下实现全球覆盖并且跟踪的项目保持在覆盖范围内时，蜂窝网络会更好。对于户外、远程和农业应用，LoRa 几乎总是在总拥有成本方面获胜。

LoRa GPS 跟踪器的准确度如何？

Loko 中的 GPS 精度完全来自 GNSS 接收器，而不是 LoRa 无线电。 Loko 同时支持 GPS、GLONASS、Galileo 和 BeiDou，在开阔天空中提供约 2-5 米 CEP 的位置精度 - 与任何多星座 GNSS 设备相同。 LoRa 无线电传输该位置而不会引入任何位置误差。

哪些因素影响 LoRa GPS 跟踪器范围？

主要因素包括：接收器的仰角（更高的仰角 = 更远的范围）、地形（视线提供最大范围，丘陵和茂密的森林会缩小范围）、扩频因子（更高的 SF = 更远的范围，但更新速度更慢）、发射功率和天线质量。在具有高架接收器的开阔地形中，Loko 的性能达到 20+ km。

LoRa GPS 跟踪器可以同时跟踪多个设备吗？

是的。 Loko 支持多个 Loko Air 跟踪器同时连接到单个 Loko Ground 接收器。所有跟踪的位置都会实时显示在同一个离线地图上，这对于通过单个设备跟踪一群动物、一群人或一队车辆非常有用。
