GPS 和 LoRa 技术指南

无线电视线、LoRa P2P 与 LoRaWAN、电离层对 GPS 的影响，以及离线 GPS 跟踪如何在没有蜂窝网络的情况下工作。

核心概念

无线电视线和信号传播

发射和接收天线之间的直接路径没有障碍物。

在 GPS 和 LoRa 技术中，视距通信对于最佳范围至关重要。无线电波在障碍物周围会稍微弯曲，但较大的障碍物会显着降低信号强度。

Loko GPS 跟踪器使用 LoRa 技术，在理想视距条件下可实现长达 20 公里的范围。

电离层和 GPS 信号质量

影响 GPS 信号质量和准确性的地球大气层。

电离层延迟可能会导致 GPS 定位错误，尤其是在太阳活动频繁期间。 Loko 使用多个星座（GPS、GLONASS、伽利略、北斗）来减轻这些误差。

了解电离层高度变化有助于优化跨区域的跟踪器性能。

LoRa 与 LoRaWAN 和 P2P 通信

LoRa是物理层； LoRaWAN 是其之上的网络协议。

P2P LoRa 创建直接的设备到设备连接，无需基础设施。 Loko 同时使用 P2P 和 LoRaWAN，以在任何环境中实现最大的灵活性。

直接 P2P 通信使其成为无需外部服务器的远程跟踪的理想选择。

技术文档

LoRa 调制解释

实现 Loko 远距离能力的技术：

扩频因子 (SF)

SF7–SF12。值越高，范围越大，但数据速率越低。 Loko 使用自适应 SF 根据信号条件平衡范围和吞吐量。

编码率 (CR)

错误修正从 4/5 到 4/8。较高的 CR 可以提高抗干扰能力。 Loko 优化了 CR，以便在充满挑战的环境中实现可靠的通信。

线性调频扩频

使用线性 FM 线性调频信号跨频率传播信号，具有出色的抗干扰性，并允许多个设备共享同一频道。
