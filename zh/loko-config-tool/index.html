Loko GPS 追踪器配置工具

如何使用 Loko 配置工具

Loko 在线配置工具让您可以直接在浏览器中使用 Web Serial API 配置 Loko Air 和 Ground 单元——无需安装软件。您需要 Google Chrome（桌面版）和一根 USB 数据线来连接设备。

分步连接指南

  1. 在桌面电脑（Windows、macOS 或 Linux）上使用 Google Chrome 打开此页面。
  2. 打开您的 Loko Air 或 Ground 单元电源。
  3. 使用 USB-C 数据线 将设备连接到电脑。
  4. 在上方工具中选择正确的标签页（Air 单元或 Ground 单元）。
  5. 点击 连接，然后在浏览器提示中选择正确的串口。
  6. 工具将自动读取设备的当前设置。
  7. 编辑任何参数，然后点击 应用 将其保存到设备。

可配置参数 — Air 单元

Air 单元是附着在您要追踪的资产上的 GPS 追踪器。可以配置以下参数：

  • 设备 ID 1 和 ID 2 — 将 Air 单元与地面站配对的唯一标识符。配对的设备之间 ID 必须匹配。
  • 无线电频率 (Hz) — LoRa 传输频率。必须与地面站匹配。默认频段：868 MHz (EU)、915 MHz (US)、923 MHz (AS)。
  • GPS 更新间隔 (秒) — Air 单元传输其 GPS 位置的频率。较短的间隔会增加电池消耗。范围：1–3600 秒。
  • 轨迹记录周期 — GPS 点保存到设备内置数据记录器的频率（最多 6,500 个点）。
  • 固件版本 — 显示设备上的当前固件。当有新版本可用时，请从 固件页面 更新。
